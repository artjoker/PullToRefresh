# Usage 
Add it in your root build.gradle at the end of repositories:

    allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
	
Add the dependency:

	dependencies {
	        compile 'com.github.artjoker:PullToRefresh:7d65ec772d'
	}
